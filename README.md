A simple React-based weather application, that fetches weather updates from OpenWeather Weather API. 

## Dependencies

- axios 0.21.1
- react 17.0.1
- react-dom 17.0.1
- react-loader-spinner 4.0.0
- react-scripts 4.0.0

## Quick start:

### Install Dependencies

```sh
$npm install
```

### Run development server

```sh
$npm run dev-server
```

### Build the project

```sh
$npm run build
```
